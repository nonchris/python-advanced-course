\noindent \tb{Sieb des Erathosthenes} \quad
  
    \noindent
    In dieser Aufgabe ist es Ihr Ziel, alle Primzahlen bis zu einer gegebenen Zahl \pp{N} zu berechnen.\\
    Das Sieb des Erathosthenes ist ein simples, aber effizientes Verfahren, zu Ermittlung dieser.\\
    Das Ziel ist es, am Ende ein Array zu haben, welches an allen Primzahl-Indizes \pp{True} enthält, und überall sonst \pp{False} ist.\\

    Die folgenden Schritte müssen getan werden:
    \begin{enumerate}
        \item Erstellen Sie zuerst ein \pp{numpy}-Array der Länge \pp{n+1}, welches komplett mit \pp{True} oder \pp{1} gefüllt ist.
        \item Setzen Sie die Elemente an den Indizes \pp{0, 1} auf \pp{False}, da 1 und 0 keine Primzahlen sind.

        \item Setzen Sie alle Vielfachen des Wertes 2 auf \pp{False}.
        \item Führen Sie den selben Schritt für alle Indizes $3\leq i \leq \sqrt{n}$ durch, bei denen ein \pp{True} in dem Array steht, da Vielfache eines Wertes niemals eine Primzahl sind.
        \item Alle nun noch auf \pp{True} stehenden Indizes sind Primzahlen.
        \item Finden Sie einen eleganten Weg, mit Hilfe dieses Boolschen-Arrays nun die Indizes bzw. repräsentierenden Zahlenwerte dieser Stellen zu berechnen.
        \item Geben Sie das Ergebnis aus.  
    \end{enumerate}
    Hinweis: Sie brauchen genau eine \pp{for}-Schleife, um die Aufgabe zu lösen. Alle anderen Operationen lassen sich durch NumPy realisieren.

    \noindent \tb{Echte Datenanalyse} \quad
      
        \noindent
        In dieser Aufgabe sollen Sie einen echten Datensatz auswerten. Hierzu haben Sie den Datensatz \pp{boulders\_habitat\_bonn.csv} und \pp{boulders\_habitat\_beuel.csv}\footnote{Die 2023er Versionen haben nur die halbe Größe und sind daher ggf. für langsame Systeme besser geeignet.} erhalten. Dieser enthält die Daten für die Auslastung der beiden Bonner Boulder Hallen von Mai 2022 bis heute.\\

        \noindent Hinweis: Es kann Sinn machen, diese Aufgabe in einem Jupyter-Notebook zu lösen.

        \begin{enumerate}
            \item Machen Sie sich mit \pp{read_csv}\furl{https://pandas.pydata.org/docs/reference/api/pandas.read_csv.html} von pandas vertraut und erstellen Sie einen der beiden Datensätze in einen \pp{DataFrame} aus der gegebenen CSV Datei.\\
            Folgende Schritte können Sie direkt beim Einlesen erledigen, oder im Nachhinein tun:

            \item \begin{itemize}
                \item Konvertieren Sie die erste Spalte in \pp{Integer} und die zweite in \pp{datetime} Objekte.
                \item Setzen Sie die \pp{datetime}-Spalte als Index.
                \item Die dritte Spalte wird nicht benötigt.
            \end{itemize}

            \item Erste Analyse:\\
            Nutzen Sie \pp{.describe()}, um sich einen ersten Eindruck über den Datensatz zu machen.
            
            
            \item Der Datensatz ist mit einer Dichte von einem Datenpunkt alle fünf Minuten ziemlich mächtig.
            Bilden Sie daher den Average über je eine Stunde.\\
            Dies können Sie mit \pp{.resample("30min").mean()}\furl{https://pandas.pydata.org/docs/reference/api/pandas.DataFrame.resample.html\#pandas.DataFrame.resample} tun.

            \item Führen Sie die selben Schritte für den zweiten Datensatz durch.
            
            \item Fügen Sie die beiden nach Zeit gruppierten Datensätze zusammen.
            
            \item[] Sie können alle nachfolgenden Aufgaben entweder für beide Hallen oder nur die Halle Ihrer Wahl auswerten.

            \item Welche halbe-Stunde(n) hatten insgesamt die höchste Auslastung? Waren es die selben Zeiten in Bonn und Beuel?
            
            \item Extrahieren Sie die Daten für Ihren Geburtstag. Wie hoch war die Aulastung an diesem Tag pro Stunde?\\
            \item[] Plotten Sie den Verlauf des Tages.
            \item[] Fragen Sie sich nun, warum all diese Leute nicht auf Ihrer Geburstagsfeier waren.
            
            \item Nachdem Sie die mittelschwere Weg-Krise überwunden haben: Gibt es eine Stunde, welche im Durchschnitt die geringste Auslastung hat?
            
            \item Gibt es Tage für welche keine oder nur wenige Datenpunkte vorliegen?\\
            Gruppieren Sie die nicht zusammengefassten Daten nach Tag und zählen Sie die Einträge pro Tag.\\
            Hinweis: Sie können mit \pp{.date} nur auf das Date Attribut eines \pp{datetime}-Objekts zugreifen.
            
            \item Gruppieren Sie die Daten nach der Uhrzeit des Indexes.\\
            Iterieren Sie über die Zeitslots und berechnen Sie die durchschnittliche Auslastung. Welcher Zeitslot ist am beliebtesten?\\
            Passt diese Beobachtung mit Ihrer Beobachtubng zur halben Stunde mit der höchsten Auslastung zusammen?\\
            Hinweis, Sie kommen mit \pp{.time} nur an das Zeitattribut eines \pp{datetime}-Objekts.
            
            \item Erstellen Sie einen Bar-Plot, welcher die gemittelte Auslastung jeder halben Stunde darstellt.
            
            \item Fallen Ihnen weitere spannende Analysen ein? Lassen Sie Ihrer Kreativität freien Lauf! Wie wäre es z.B. mit einem Vergleich zwischen den Jahren?

        \end{enumerate}
